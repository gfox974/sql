MYSQL : langage de requête pemrttant d'intéroger une base de données

4 requete SQL :
INSERT => inseriton
UPADTE = > modification
DELETE = => suppresion
SELECT => séléction


Quelque termes : 
    MCD : Modèle conceptuel de données (plan de construction)
    SGBD : système de gestion de bas de données (exmploite les donées)
    BDD : Bas de Donnés (Emplaceme,nt de donéées sauvegarder)
    SQL : structured query language (langage de requêtes )

Outils nécéssaire (serveur local) : xampp | wampp | laragon ou autre...
